# Employee Salary Calculator

A lightweight Java console application built to practice OOP fundamentals through salary calculation logic.

A Java application that calculates a factory employee’s financial details, including tax deductions, bonus payments, and salary raises based on years of service. This project demonstrates clean OOP design, method structuring, and formatted output generation.


## Features
- Calculates tax based on salary thresholds

- Computes weekly bonus for overtime work

- Determines salary raise based on years worked

- Generates a detailed financial breakdown using toString()

- Clean and readable object oriented structure

### Project Structure
- Employee.java — Core class containing salary logic

- Main.java — Entry point that creates an employee and prints results
